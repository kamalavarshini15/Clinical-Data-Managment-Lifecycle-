# Clinical-Data-Managment-Lifecycle-
This repository contains the overall process of CDM - Step by Step work flow for easy understanding .
Focuses on Main activites in CDM.



**MAIN ACTIVITES IN CDM PROCESS**

***1) Study Start up 
2) Study Conduct 
3) Study Close out*** 


Study start Up:
> Study Design
> eCRF Design
> Edit checks
> Derivation checks
> UAT testing


Study Conduct: 
> Data Entry
> Data Validation
> Data Coding
> SAE Reconciliation
> Lab Reconciliation

Study Close out:
> Database Lock
> Statistcial Report generation
